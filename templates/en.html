  • Province

Use Case 1: Get Address

One popular use case in applications is get user addresses. This task would become easy by using VN Provinces API.

Get address step by step

Getting user addresses through steps:

  • Get provinces list, then users select their provinces
  • After got province, load districts of provided province
  • Similary, load wards after got district.

To do above steps, this is how to call API:

  • Get provinces:
    • [[getUrl(0)]][[getQuery(0)]]
    • Parameter basic=true to get province with basic information.
    • Parameter limit=100 to get all results without pagination
  • Loading districts of a provided province. There are 2 ways to do:

    • [[getUrl(3)]][[getQuery(3)]]

Get address by search ward name

Above method introduces how to get address by enter all information from province, district then ward. There is another way to achieve the same result but only need to search ward name (and district if neccessary).

  • Input ward and district (separate by a comma)

How to call API:

    [[getUrl(0)]][[getQuery(0)]]
    Code is copied to clipboard!

Api will return wards with the search keyword, along with their districts and provinces.

In most scenarios, we only need to enter ward to get full address. In case of there are many return results, we would need to filter them by district.

Use case 2: Search provinces, regions, districts, and wards

The second application of API is to search, filter and sort information about administrative units in Vietnam

By apply search keywords, parameters, API will return right results

Please see examples below:

  • Fill all the Central Cities in Vietnam (jurisdiction of central goverment)
    • [[getUrl(0)]][[getQuery(0)]]
  • Fill 5 largest provinces in Vietnam
    • [[getUrl(1)]][[getQuery(1)]]

    Use parameter ordering=-area to sort province in descending order. Because the default order is ascending, so use - before parameter area to inverse the order. We only need 5 results, so use limit=5.

  • Which province does number plate 50 belong to
    • [[getUrl(2)]][[getQuery(2)]]
  • Among Hanoi neighbour provinces, which one is most populous
    • [[getUrl(3)]][[getQuery(3)]]
  • Do you know any districts in Vietnam which is both coastal and border?
    • [[getUrl(4)]][[getQuery(4)]]
  • It's interesting that in Vietnam, there are some districts which no child ward.
    • [[getUrl(5)]][[getQuery(5)]]
  • Fill all regions in Vietnam and sort by the number of its provinces (descending order)
    • [[getUrl(6)]][[getQuery(6)]]
